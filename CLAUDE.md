# Project: Bragas Marta - E-commerce Landing Page

## Overview
Static e-commerce landing page (HTML + CSS + JS). No build step, no frameworks.

## Key Files
- `index.html` - Main page
- `styles.css` - All styles
- `app.js` - Interactive functionality
- `img/` - Product images (JPG + SVG)

## Deployment
- **Hosted on GitHub Pages**: https://elbenxo.github.io/proyecto-web-m/
- **Method**: Deploy from branch (NOT GitHub Actions — Actions is disabled on this account)
- **Deploy branch**: `gh-pages` (with `.nojekyll` file to skip Jekyll processing)
- **To deploy changes**: merge `main` into `gh-pages` and push, or push directly to `gh-pages`

## Git
- **Main branch**: `main` (development)
- **Deploy branch**: `gh-pages` (production)
- Changes on `main` are NOT automatically deployed — you must update `gh-pages` to deploy
